# Learnify

Open-source, theory-first coding tutorials (W3Schools-style). React + JSON served live from the server — no login, no PWA, no offline mode (internet required).

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Topics

HTML, CSS, JavaScript, Python, SQL, Java, DSA, and React — lesson source files live in `src/content/*.json` and are copied to `public/content/` at dev/build time, then fetched over the network at runtime (not bundled for offline use).
